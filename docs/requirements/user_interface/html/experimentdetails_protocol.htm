Protocol Name:

Protocol A
Description:

Analysis of tissues and associated data for protein and/or gene expression and/or DNA sequencing studies - Breast Cancer Protocol. This is a tissue study to obtain breast cancer specimens for proteomic analysis in real time (door to door within 4 hours). The objective is to define molecular protein targets, present in cancer tissue and absent in the normal tissue.
