A generalized Actor intended to represent those applications/systems that programmatically communicate with protExpress. Examples of external systems we know who want to interact with protExpress include CPAS (Computation Proteomics Analysis System).

Examples of programmatic access might include an API, web service, or grid -- specific mechanisms that will be supported have not been solidified.
