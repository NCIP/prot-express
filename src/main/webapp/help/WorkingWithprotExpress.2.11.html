Adding Protocols to an Experiment
You can add a protocol to an experiment by creating a new one or by selecting an existing protocol. You can add a protocol to an experiment while creating a new experiment or later, while editing an experiment.
A protocol is a reusable entity that you can use across mutliple experiments. However, an instance of a protocol applied to an experiment run is unique and you cannot apply it to other runs or experiments. Each time you add an existing protocol to an experiment, you must add an input and output to complete it.
A complete experiment includes at least one complete protocol. A complete protocol includes at least one input and one output.
To add a protocol to an experiment
1.
Begin creating a new experiment. For more information on creating an experiment, see Creating a New Experiment. The Add Protocols to Experiment page, Add New Protocol tab appears.
Or, to add a protocol to an existing experiment, open the experiment run to which you want to apply the new protocol. Open an experiment run by first selecting the experiment on the protExpress home page or search results, then selecting the run in the experiment navigation on the left side of the page, and then clicking the Edit Experiment button.
