Registering Users
Your system administrator can manage user registration by validating against the database stored in protExpress or LDAP.
If the system administrator chooses to use the protExpress database, which is the default, the protExpress home page includes links to a registration page.
This section explains how to add a user account to the protExpress database.
To register to use protExpress
1.
Click the Register menu in the upper left of the home page or the Register link in the protExpress Login area. The Registration page appears.
2.
In the Username field, enter a name for the user that is unique within the local database and follows your standard naming conventions. This is a required field.
