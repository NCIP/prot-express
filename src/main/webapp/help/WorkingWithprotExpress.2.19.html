Deleting an Experiment
You must first search for an experiment before you can delete it.
To delete an experiment
1.
Search for the experiment you want to delete. For more information on searching, see Searching protExpress. The Search Results page appears.
2.
In the row corresponding with the experiment you want to delete, click . A message appears confirming that you have deleted the experiment.
