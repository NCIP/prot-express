Applying Protocols to Experiments
You can set up an experiment to apply one or more protocols in the order you specify. Each experiment run you add to this experiment will apply the same protocols in the same order.
Whenever it does not conflict with the protExpress user interface, this documentation refers to the protocol to apply as a protocol application.
You can define one or more global inputs available to each protocol you apply to the experiment. For more information, see Working with Global Inputs.
