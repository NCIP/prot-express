Adding Protocols to an Experiment
You can add a protocol to an experiment by creating a new one or by selecting an existing protocol. You can add a protocol to an experiment while creating a new experiment or later, while editing an experiment.
A protocol is a reusable entity that you can use across mutliple experiments. However, an instance of a protocol applied to an experiment run is unique and you cannot apply it to other runs or experiments. Each time you add an existing protocol to an experiment, you must add an input and output to complete it.
A complete experiment includes at least one complete protocol. A complete protocol includes at least one input and one output.
To add a protocol to an experiment
1.
You can add a protocol to a new experiment or to an existing experiment run.
When you first create an experiment and add a protocol to it, you are adding that protocol to the experiment’s first experiment run.
For more information on creating an experiment, see Creating a New Experiment. For more information on editing an experiment run, see Editing an Experiment Run.

When you add a protocol to a new experiment, you begin from the Add Protocols to Experiment page, Add New Protocol tab.
When you add a protocol to an existing experiment run, you begin from the Add New/Existing Protocol page.
2.
To add a new protocol to the experiment, do the following:
a.
Click the Add New Protocol tab if it is not already selected.
b.
In the Name field, enter a name for the new protocol. This is a required field.
c.
Enter the protocol’s description and its associated software, associated instrument, and contact information for the protocol owner in the respective fields. Entering information in these fields is optional. Note that you can use the content of the Name and Description fields in a search for this protocol later.
d.
Click . The protocol is created.
3.
To add an existing protocol to the experiment, do the following:
a.
Click the Select Existing Protocol tab. All of the protocols you created appear in the list. To see all available protocols, click the Display Protocols created by all users checkbox.
b.
Find the protocol you want to select and click . The protocol appears in the Experiment Navigation.
