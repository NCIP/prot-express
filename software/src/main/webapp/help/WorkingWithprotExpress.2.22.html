Editing Experiment Details
As an experiment owner, you can edit any of the experiment’s properties. Experiment properties in protExpress mirror those specified in the XAR format used by CPAS.
To edit an experiment’s details
1.
Open the experiment you want to edit using either of the following methods:
•
In the My Recent Experiments area on the Dashboard, click the Edit () button in the row corresponding with the experiment you want to edit.
•
After searching for an experiment (for more information, see Searching protExpress), click the Edit () button in the row corresponding with the experiment you want to edit.
2.
The Edit Experiment Details page appears.
3.
As needed, edit the experiment’s details in the text boxes on this page. Refer to the following table for more information about these details. .

When you have finished editing the experiment’s details, click . A message indicating that the experiment has been successfully updated appears at the top of the page.
