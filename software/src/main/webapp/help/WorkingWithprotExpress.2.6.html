Using the Home Page
After logging in to protExpress, you arrive at its home page. You can do the following from this page:
•
Create a new experiment
•
Search protExpress for any experiment or protocol so that you can view, edit, or download it
•
Check the status of a recently added experiment
•
Edit a recently added experiment
•
Download a recently added experiment in XAR format
To return to the home page after working in other areas of protExpress, click HOME in the main protExpress menu on the left.
