Using the Dashboard
After logging in to protExpress, you arrive at the Dashboard. To return to the Dashboard after working in other areas of protExpress, click Dashboard in the protExpress menu.
The protExpress Dashboard lists the three protocols and three experiments that you must recently added to protExpress, along with the dates when you added them. Only protocols and experiments that you own appear in these lists.
You can start the following tasks from the Dashboard:
•
View a protocol or an experiment
•
Search for a protocol or an experiment to view
•
Add a new protocol or an experiment
•
Edit a protocol or an experiment
•
Import experiment data
