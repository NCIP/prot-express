Adding an Existing Protocol to an Experiment
To add an existing protocol to an experiment
1.
Click the Select Existing Protocol tab. All of the protocols you created appear in the list. To see all available protocols, click the Display Protocols created by all users checkbox.
2.
Find the protocol you want to select and click . The protocol appears in the Experiment Navigation.
A complete protocol requires at least one input and one output. You can add them at any time by editing the protocol. For more information, see Editing Protocol Details, Adding an Input, and Working with Outputs.
